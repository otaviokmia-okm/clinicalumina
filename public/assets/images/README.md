# Ativos Lumina Aesthetics

Utilize esta pasta para organizar as imagens personalizadas da clínica.

## Como utilizar:
1. Coloque suas imagens aqui (ex: `logo.png`, `hero.jpg`).
2. No arquivo `src/app/lib/placeholder-images.json`, mude o campo `imageUrl` para o caminho relativo: `/assets/images/sua-imagem.jpg`.
