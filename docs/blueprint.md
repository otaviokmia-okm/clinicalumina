# **App Name**: Lumina Concierge

## Core Features:

- Luxury Service Presentation: Visually stunning display of Lumina's aesthetic services with high-quality descriptions and imagery, incorporating hero section parallax and elegant service grids.
- Refined Appointment Scheduling: An intuitive, modern calendar widget allowing clients to select services, choose from available time slots, and securely submit personal details for booking, starting with a 'Pendente' status.
- Admin Appointment Dashboard: A secure administrative panel where staff can review, manage (confirm, reschedule, cancel), and track all client appointments stored in Firestore, with visual status indicators.
- Robust Admin Authentication: Secure login mechanism for authorized access to the administrative dashboard, leveraging Firebase Authentication.
- Adaptive User Interface: Ensures full responsiveness across desktops, tablets, and mobile devices, utilizing Tailwind CSS for sleek adaptation and consistent premium user experience.
- Instant Client Engagement: A discreetly animated, pulsing WhatsApp button fixed in the bottom-right corner for immediate client inquiries and direct communication.
- AI-Enhanced Confirmation Tool: Generates personalized confirmation messages or pre-treatment guidance based on the client's booked service, elevating the booking experience with bespoke communication.

## Style Guidelines:

- A sophisticated light color scheme. The primary color is a deep, muted bronze (#8A816D), embodying quiet luxury and depth, ideal for key accents and typography that demands attention against a light backdrop. The background color is a warm, soft off-white (#F4F2EE), offering an airy and expansive feel. The accent color is a muted rose gold (#BA948B), analogous to the primary, providing a subtle, refined visual emphasis.
- Headlines and titles will feature 'Alegreya' (serif), chosen for its elegant, intellectual, and contemporary feel, lending gravitas to content. Body text and supplementary information will use 'Inter' (sans-serif), a modern, objective font ensuring readability and a sleek aesthetic. This pairing balances classical luxury with modern clarity.
- Utilize thin-line, minimalist icons from Lucide-react or FontAwesome to maintain a clean interface that complements the sophisticated typography and overall quiet luxury aesthetic.
- Clean, minimalist layouts with generous whitespace to emphasize content and photography. The hero section features an engaging parallax effect, while services are presented in elegant grid cards. Navigation is intuitive, contributing to an effortless user journey on all device sizes.
- Subtle and graceful entrance animations for key elements using Framer Motion, ensuring a polished and smooth user experience. The WhatsApp button will feature a gentle, pulsing animation to invite engagement without being intrusive.
